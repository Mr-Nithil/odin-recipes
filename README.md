# Odin Project - Foundation Course

Recently I have started working on The Odin Project to learn full stack development from the scratch level. Even though I have experience in web development, I decided to take a step back and started a 3-month run to focus on real web development from the ground up. The Odin Project is a free, open-source, self-learning platform. So far I have started the journey and gone through the below topics. Let's see what I will achieve at the end of these 3 months!

- **How this course will work**
  - Learned how The Odin Project's curriculum works.
  - Learned how the Foundations course works.
- **Introduction to Web Development**
  - Learned about different types of web developers.
  - Recognized different types of clients and employers.
  - Learned about tools used regularly in development.
- **Motivation and Mindset**
  - Recognized sensible mindsets that promote healthy learning.

  - Learned how to ask good and detailed questions.
  - Learned good practices for helping others with their questions.
